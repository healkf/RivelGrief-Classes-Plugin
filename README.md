RivelGrief Classes Plugin - это плагин разрабатываемый для всеми нами известного сервера. Суть в том, что человек при первом заходе на сервер может выбрать себе класс, 
с которым будет играть на сервере. остальные классы нужно открывать выбивая их на ивентах и открывая лутбоксы
Плагин всё еще не дописан и требует доработок, но в целом он уже довольно играбельный и приятный.

Чтобы установить класс игроку напишите команду: /setclass <игрок> <класс>
Чтобы разблокировать закрытый класс для игрока напишите команду: /unlockclass <игрок> <класс>
Открыть меню выбора класса (сам/другому игроку): /classmenu (игрок)

При выборе класса также возможна выдача предмета этого класса, так сказать ключевая фишка. Этот предмет нельзя выбросить.
